# Western Astrology Skill

Use this skill for Western astrology users routed by international phone prefixes.

## Core Commands

Calculate a tropical Western natal chart:

```bash
python3 ~/.openclaw/skills/western/natal_chart.py --dob "1990-08-15" --tob "14:30" --place "New York" --json
```

Generate a chart wheel image:

```bash
python3 ~/.openclaw/skills/western/draw_natal_chart.py --chart-json '<JSON_FROM_NATAL_CHART>' --user-id "<USER_ID>"
```

Trigger a backend-generated PDF report:

```text
WESTERN_PDF_REQUEST: dob=YYYY-MM-DD, tob=HH:MM, place=CITY, name=NAME
```

Get built-in interpretations:

```bash
python3 ~/.openclaw/skills/western/western_interpretations.py summary --sun Leo --moon Scorpio --ascendant Libra
```

Get remedies:

```bash
python3 ~/.openclaw/skills/western/western_remedies.py remedies Leo
```

## Accuracy Rules

- Birth time is treated as local birth-place time.
- `natal_chart.py` converts local birth time to UTC before Swiss Ephemeris calculations.
- If Swiss Ephemeris is unavailable, only the simple Sun sign fallback is reliable.
- If timezone cannot be detected, the tool emits a warning. Mention the uncertainty briefly.
- Use tropical zodiac and Western houses. Do not mix Vedic terms like Lagna, Nakshatra, Dasha, Kundli, Rahu, or Ketu unless the user explicitly asks for a comparison.

## Media Output Rules

The image tool prints either:

```text
IMAGE_URL: <stored-image-url>
```

or:

```text
MEDIA_BASE64: image/png <base64>
```

For PDFs, use `WESTERN_PDF_REQUEST:`. The WhatsApp backend generates and sends the document.

Send image media exactly as generated by the platform. Do not paste base64 text to the user.

## Workflow

1. Fetch Mem0 and MongoDB conversation history first.
2. If birth details are already known, run `natal_chart.py`; do not ask again.
3. Search `western_astrology` Qdrant for the user's topic.
4. Combine chart placements, Qdrant knowledge, memory, and the user's question.
5. Save important birth details, chart placements, and reading summaries to Mem0.
6. Keep WhatsApp replies short, warm, and human.
